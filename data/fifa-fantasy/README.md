# FIFA Fantasy Data

Fetched from FIFA's public fantasy static JSON endpoints:

- `https://play.fifa.com/json/fantasy/players.json`
- `https://play.fifa.com/json/fantasy/squads.json`
- `https://play.fifa.com/json/fantasy/rounds.json`

`players.csv` is an enriched export for draft tooling. It joins each player to
their squad and scheduled next fixture.
